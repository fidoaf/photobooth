Fun online webcam photobooth powered by p5js

## Available filters
- No effect
- Upside down
- Bottom mirror
- Top mirror
- Left mirror
- Right mirror
- Double top
- Double bottom
- Threshold
- Gray
- Opaque
- Invert
- Posterize
- Dilate
- Blur
- Erode
- Colorize
- Noise
- Color bars
- Dotted
- Pixelate
